Docs Site Prototype

Inject your reimagined Products section into the live docs.datadoghq.com homepage to test it in context.

Inject Docs Prototype
Drag this to your bookmarks bar, or right-click → Bookmark this link

How to use

1
Make sure the prototype dev server is running:
cd ~/Downloads/docs-site-prototype && npm run dev
2
Drag the purple button above to your bookmarks bar.
3
Navigate to docs.datadoghq.com.
4
Click the bookmarklet. The Products section is replaced with your prototype.
5
Resize the browser to test responsiveness. Use the floating Restore Original button to toggle back.
Tip: Click the bookmarklet again to toggle between the original and your prototype. The prototype toolbar (top-right corner) lets you tweak descriptions, scaling, and more.
